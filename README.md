# Luntian

This project now uses PHP as its entry point.

## Run locally

1. Open a terminal in the project root.
2. Start the PHP development server:

```bash
php -S localhost:8000
```

3. Open `http://localhost:8000` in your browser.

The app entry file is `index.php`, while styling and behavior stay in `styles.css` and `main.js`.

## Available routes

- `index.php?page=home`
- `index.php?page=start`
- `index.php?page=poems`
- `index.php?page=settings`
